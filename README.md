# Retail Sales Analysis and Reporting Dashboard

##  Project Overview
This project focuses on analyzing retail sales data and generating meaningful insights using Python, SQL, and Excel.  
An interactive dashboard is developed using Streamlit to visualize trends and support decision-making.

---

##  Objectives
- Analyze retail sales data
- Perform data cleaning and preprocessing
- Generate insights using SQL and Excel
- Build an interactive dashboard using Streamlit
- Automate reporting and visualization

---

##  Tools & Technologies
- Python (Pandas, Plotly, Streamlit)
- SQL (SQLite)
- Excel (Automated Dashboard)
- ReportLab (PDF Generation)

##  Features

###  Interactive Dashboard
- Filters (Country, Month, Date)
- Search functionality
- KPI metrics (Sales, Orders, Countries)
- Visualizations (Bar, Pie, Line charts)

###  SQL Analysis Panel
- Displays SQL queries and results
- Covers:
  - Total Sales
  - Sales by Country
  - Top Products
  - Monthly Trends

###  Dynamic Report
- Includes:
  - Abstract
  - Introduction
  - Objectives
  - Data Cleaning
  - Results & Findings
  - Conclusion
- Updates automatically based on selected filters
- Downloadable as PDF

###  Excel Dashboard (Automated)
- Generated using Python
- KPI-based layout
- Insights summary
- Structured dashboard format
